# Windows 电脑网络基础设置
## 一、查看本机网络信息
1. 使用快捷键 `Win + R`，输入 `cmd` 打开命令行
2. 输入 `ipconfig` 查看本机 IP、子网掩码、网关、DNS
3. 无线 IP 一般在 WLAN 栏目，有线 IP 在以太网栏目

## 二、连接 WiFi
1. 点击桌面右下角网络图标
2. 选择自家 WiFi 名称，输入密码连接
3. 无法连接时：点击“忘记此网络”后重新连接

## 三、切换自动获取IP / 手动设置IP
1. 打开 `设置 → 网络和 Internet → 以太网/WLAN`
2. 点击 `编辑 IP 设置`
3. 自动获取（推荐）：选择 `自动(DHCP)`
4. 手动设置：选择 `手动`，开启 IPv4，填写
   - IP 地址：如 192.168.1.100
   - 子网前缀：24（对应 255.255.255.0）
   - 网关：192.168.1.1
   - DNS：114.114.114.114

## 四、修改 DNS 解决网页打不开
1. 进入网络适配器设置
2. 右键当前网络 → 属性 → 双击 `Internet 协议版本 4 (TCP/IPv4)`
3. 选择 `使用下面的 DNS 服务器地址`
4. 首选 DNS：114.114.114.114
5. 备用 DNS：8.8.8.8

## 五、启用/禁用网卡
1. 右键开始菜单 → `网络连接`
2. 右键 WLAN 或以太网
3. 选择 `禁用`，再 `启用` 可快速修复网络故障

## 六、释放并重新获取 IP（修复 IP 异常）
以管理员打开 cmd，依次执行：
1. `ipconfig /release` 释放当前 IP
2. `ipconfig /renew` 重新获取 IP

## 七、刷新 DNS 缓存（解决域名解析错误）
cmd 中执行：
`ipconfig /flushdns`

## 八、关闭/打开 Windows 防火墙
1. 打开 `控制面板 → 系统和安全 → Windows 防火墙`
2. 选择 `启用或关闭 Windows 防火墙`
3. 可临时关闭测试网络，正常使用建议保持开启

## 九、查看正在联网的程序与端口
1. cmd 输入 `netstat -ano`
2. 查看本机哪些程序正在占用网络、连接外部服务器

## 十、代理服务器关闭（解决莫名上不了网）
1. `设置 → 网络和 Internet → 代理`
2. 关闭 `使用代理服务器` 开关
3. 很多无法上网问题是代理异常导致
